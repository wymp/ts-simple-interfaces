SqlPrimitive

SqlPrimitive: string | number | boolean | BufferLike | Date | null
  • Defined in src/index.ts:255

While any SQL datasource may be easily implemented as a SimpleDatasource, they may also be implemented more specifically as a general SimpleSqlDbInterface. This interface is intended to help unify the various implementations of SQL databases out in the wild, such that they may be more plug-and-playable.

SimpleHttpServerNextFunction

SimpleHttpServerNextFunction: (errOrOtherVal?: any) => void
  • Defined in src/index.ts:481

Type declaration

    • (errOrOtherVal?: any): void

    • This function is passed by the Webservice to all handlers and is responsible for chaining middleware together and for handling errors generated by handlers. In the event that an error occurs while processing a request, you can pass that error to this next function to activate any error handling logic you've put in place. Otherwise, control is simply passed to the next middleware in the stack.
